# Notes frontend Project

## Development server

1. Up Backend on 4005 port
2. Use VITE_BACKEND="http://localhost:4005" in .env
3. Use yarn
4. Run `npm run dev -- --port 3007` for a dev server. Navigate to `http://localhost:3007`. The application will automatically reload if you change any of the source files.

## Docker server

1. Use VITE_BACKEND="http://backend:4005" in .env
2. Go to folder root
3. docker compose up --build -d
