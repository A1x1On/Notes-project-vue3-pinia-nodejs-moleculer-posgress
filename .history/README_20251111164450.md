# NOTE Project

Мини проект заметок vue3/typescript/tailwindcss/pinia/nodejs/moleculer/typeORM/postgres and others
- Docker build
- CRUD logic

## Docker Server
1) Go to folder root
2) docker compose up --build -d

## Docker Server (Restore database)
1) Go to folder root
2) docker compose down --volumes --remove-orphans
3) docker compose up --build -d
4) if it need docker compose up -d

- Отдельные настройки для frontend сборки можно найти ./frontend/README.md
- Отдельные настройки для backend сборки можно найти ./backend/README.md
